Transcrição de áudio e vídeo

Faça upload de um arquivo MP3, WAV, M4A, MP4, MOV ou WebM e gere a transcrição. Esta página oferece dois modos:

Modo 1: API da OpenAI (mais confiável) Modo 2: Navegador local com Whisper tiny (mais leve, porém mais limitado)
Vídeos são aceitos. O arquivo é enviado para a API apenas se você escolher o modo OpenAI.
A chave fica só no seu navegador. Ela não é salva automaticamente.
Use para ajudar em nomes próprios ou contexto específico.

Neste modo, o processamento ocorre no navegador usando Transformers.js. A primeira carga pode demorar e baixar centenas de MB do modelo. Em celulares ou navegadores mais fracos pode falhar.

Aguardando arquivo.
Você pode editar o texto manualmente depois.
